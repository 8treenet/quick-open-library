## quick-open-libray
### vscode市场中搜搜 quick-open-libray

#### 安装code 命令到环境变量
```
1. command + shift + p
2. 输入 > code
3. 选择输入回车 > Shell command 'code' successfully installed in PATH.
```

#### 效果展示
##### 跳转到依赖的项目文件，右键菜单点击Quick open， 会以当前依赖文件的目录打开新的工程。
